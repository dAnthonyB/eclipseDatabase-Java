Program is made to run eclipse.
AccountView contains the main method.
It will try to automatically load the save file from "testfileHW4.txt," 
but you can also enter the filename as a command-line argument.
If the .ser files don't load, it will generate a predefined file.
When run, a menu will appear with commands that have the required functionality.
User IDs are generated by the system automatically starting at 0.
